Linkedin User Dump
Use of this extension is against LinkedIn TOS and your account may be restricted.
  1. Sign into LinkedIn
  2. Browse to the required company page
  3. Click 'See all ### employees on linkedin'
  4. If there are more than 1000 results (100 pages) then use the filters to reduce the number and run multiple dumps
  5. Click the Dump Users button below and wait for the message box
  6. If you need to cancel the dump then close the LinkedIn tab

Options:
Remove junk from names
Generate possible usernames (Forces junk removal)
